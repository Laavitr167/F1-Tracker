# F1-Tracker 🏎️

A mobile app for tracking Formula 1 race data, built with React Native and Expo.

## About 
This is my first mobile development project — built as a way to learn React Native
and Expo while working with something I'm genuinely into: Formula 1.

## Tech Stack
- React Native
- Expo

## Status
🚧 Actively in progress. This is a separate, earlier project from my main upcoming
work with FastF1/Python for race telemetry analysis.

## Roadmap
1 - Pull live race data from an F1 API
2 - Improve UI/UX
3 - Add driver and team stats

## Getting Started
```bash
npm install
npx expo start
```
